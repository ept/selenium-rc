A short explanation about the directory structure of this project.
trunk/code/
    The source for Selenium itself.
trunk/website/
    The source for the Selenium website mounted for live at http://selenium.thoughtworks.com
trunk/code/javascript/
    This is the In-browser version of Selenium.
trunk/code/java/
    This is the Java version of the Driven Selenium. Ant is required to compile it, run unit and selenese tests. It leverages the trunk/code/javascript/ directory too.
trunk/code/python/
    This is the Python version of the Driven Selenium. It leverages the trunk/code/javascript/ directory too.
trunk/code/ruby/
    This the Ruby version of the Driven Selenium. It leverages the trunk/code/javascript/ directory too.
trunk/code/dotnet/
    This is the .Net version of Driven Selenium. It requires Nant. It leverages the trunk/code/javascript/ directory too.
