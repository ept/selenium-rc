A short explanation about the directory structure of this project.
trunk/
    The root. Contains a top-level Ant script that builds everything, as well as a top-level Maven POM that builds all the other Maven projects below (not everything is Maven-ized)
trunk/javascript-core/
    "Selenium Core", includes the JS libraries and HTML files. It includes a Maven POM file so it can be repackaged for other Maven builds.
Note that to run this Maven build, you first have to do your Maven build from /trunk.
trunk/server/
    The source for the Selenium RC Server, which is only in Java, and requires Maven to build; depends on Core, and repackages it so other builds won't depend on Core.
trunk/clients/
    The source for the Selenium RC Clients - java/, dotnet/, python/, ruby/.
trunk/clients/java/
    The source for the Java RC Client. Requires Maven to build, depends on Server.
trunk/clients/dotnet/
    The source for the .NET RC Client. Requires Ant and Visual Studio to build; depends on a successfully built Server to run its integration tests.
trunk/clients/python/
    The source for the Python RC Client. Requires Ant and Python; depends on a successfully built Server to run its tests.
trunk/clients/ruby/
    The source for the Ruby RC Client. Requires Ant and Ruby; depends on a successfully built Server to run its tests.
trunk/website/
    The source for the Selenium RC website, which is exposed live
