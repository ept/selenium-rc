Stating the obvious

Konqueror, is a web browser that was created for the KDE environment on the Linux Operating System. You will not find it on Windows or Mac OS X.

And the not so obvious

There are many flavors of Linux and konqueror could be installed almost anywhere and still work as prescribed. Unfortunately Selenium-RC needs to know where Konqueror is installed before it can launch the browser as a process.

Typically Konqueor is installed in /usr/bin/konqueror. On the Linux machine where Konqueor would run, open up a Shell (Terminal) and type.. 

  ls -al /usr/bin/konqueror

You should something like:

  -r-xr-xr-x   15 root  wheel  147 Aug 19  2006 /usr/bin/konqueror

Try executing that same command from the Shell you are in. Try executing konqueror on its own, without the /usr/bin/ prefix. If that works, do a 'which konqueror' to see where it is installed.

Symlinking to the non default location.

A quick way of getting Selenium-RC to work with your non default location for Konqueror, is to make a link to that executable that you do have. Try ..

   ln -s /path/to/the/real/konqueror /usr/bin/konqueror
