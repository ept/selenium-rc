Server Command Line Options

-port: specifies the port you wish to run the Server on (default is 4444)

-timeout: specifies the number of seconds that the server will wait for the browser to respond to the previous command, or for the client driver to issue a new request.

-interactive: starts the server in Interactive mode. In this mode you can type Selenese commands on the command line (e.g. cmd=getNewBrowserSession&1=*firefox&2=http%3A%2F%2Fwww.google.com).

-htmlSuite: automatically runs a single HTML suite and saves the results to a file. See the Selenese section for more information.
