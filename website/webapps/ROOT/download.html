Standalone Server
The standalone server does not require installation on the same web server as your test application. It "stands alone" on your workstation to test remote sites. Useful for testers with limited access to the test application's source code. With an XML-RPC client library, use the standalone server to drive a supported web browser from almost any programming language.

selenium-server-0.4.0-win-setup.exe Selenium standalone server for Windows. (Tested on XP.)
